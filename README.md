## 项目简介

AI 塔罗占卜让您抽取塔罗牌，AI 给出神秘而深刻的解读。

## AI 能力

- 🎴 **随机抽牌**：22张大阿尔卡纳牌
- 🔮 **深度解读**：过去/现在/未来三牌阵
- 🌙 **神秘风格**：优雅神秘的占卜语言

## 本地运行

直接用浏览器打开 `index.html` 即可使用。
